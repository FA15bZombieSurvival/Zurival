Zurival is a multiplayer survival game that has currently been developed by a team of six students as part of a school project. Given specifications are JavaScript with NodeJS and AngularJS. We were free to use any additional technologies. Therefore our team decided to create a multiplayer game to learn more about networking and server side applications. The game idea is based on some famous .io titles like agar.io or cursors.io

The game has one easily understandable rule: Survive!

Survive multiple waves of zombies per round to earn items and experience points. Exchange and trade items with other players and sell them to earn more money.

If you are interested in the code or the process of development click "Fork me on GitHub". We developed the game in public under the GNU General Public License (Version 3) to get feedback directly from the players and interested game developers. So create an account to get started or click the play button below to start a game if you are already logged in.

At last we hope you have fun with our little game. Enjoy!
